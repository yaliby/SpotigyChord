מה מתנגן בספוטיפיי + אקורדים

מתחבר ל‑Spotify, מציג את השיר שמתנגן, וטוען תוצאת אקורדים בתוך האפליקציה באמצעות scraping.

הגדרה חד‑פעמית (בלי לגעת בקוד)

  1. צור Spotify Developer App (Dashboard של Spotify Developers).
  2. הוסף Redirect URI הבא להגדרות האפליקציה שלך:
אחרי ששמרת Client ID והוספת Redirect URI בדשבורד של Spotify — לחץ “התחבר”.

פריסה

  • נדרש שרת Node.js (הפרויקט משתמש ב‑backend עבור scraping/proxy).
  • להרצה מקומית: `npm install` ואז `npm run dev`.
  • אפשר לפרוס לכל שרת שתומך ב‑Node (לא ל־GitHub Pages בלבד).
