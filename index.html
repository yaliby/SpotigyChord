מה מתנגן בספוטיפיי + אקורדים

מתחבר ל‑Spotify, מציג את השיר שמתנגן, וטוען תוצאת אקורדים בתוך האפליקציה באמצעות scraping.

הגדרה חד‑פעמית (בלי לגעת בקוד)

  1. צור Spotify Developer App (Dashboard של Spotify Developers).
  2. הוסף Redirect URI הבא להגדרות האפליקציה שלך:
ב‑GitHub Pages אפשר לעבוד גם בלי Backend. אם תגדיר Backend חיצוני, התוצאה תהיה יציבה ומהירה יותר.

תצוגת אקורדים

לחץ על "פתח אקורדים כאן" כדי לטעון את האתר הראשון בתוך האפליקציה.

פריסה

  • GitHub Pages עובד גם בלי Backend (מצב דפדפן בלבד).
  • ליציבות גבוהה יותר אפשר להגדיר Backend API חיצוני.
  • להרצה מלאה מקומית עם Backend: `npm install` ואז `npm run dev`.
